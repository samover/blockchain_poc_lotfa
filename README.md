# HACK THE PORT
## iCapps Bill of Lading on BlockChain POC

## Getting up and running
* clonse this repo
* make sure you have `node >6.9.1`
* `yarn install`
* `npm install -g truffle`
* `node server.js`
* `truffle migrate`
* `truffle build`
* open `./build/index.html`
